## Compression Plan

All raw photos are **17-28 MB each** (4000-6000px wide). For web use:

| Use Case | Target Width | Target Size | Format |
|---|---|---|---|
| Hero backgrounds | 1920px | < 300 KB | JPEG (quality 80) |
| Service cards / thumbnails | 800px | < 100 KB | JPEG (quality 80) |
| Product shots (inline) | 1200px | < 200 KB | JPEG (quality 85) |
| Article images | 760px | < 120 KB | JPEG (quality 80) |

**We'll need a compression tool** — since there's no Node.js on this machine, options are:
1. Online tool (squoosh.app, tinypng.com) — manual but works
2. Install Node.js + sharp — automated batch processing
3. PowerShell with .NET System.Drawing — no install needed
